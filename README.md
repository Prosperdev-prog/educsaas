# EDUCSAAS - Gestion Scolaire Intelligente 🚀

**Educsaas** est une solution SaaS (Software as a Service) complète conçue pour moderniser la gestion des établissements scolaires. Elle permet aux écoles de gérer leurs élèves, notes, présences et bulletins, tout en offrant un accès en temps réel aux parents.

## ✨ Fonctionnalités Clés

### 🏫 Pour l'École (Admin)
- **Gestion Multi-écoles** : Isolation totale des données par établissement.
- **Tableau de Bord** : Statistiques en temps réel sur les effectifs et les performances.
- **Gestion des Classes & Matières** : Configuration flexible du programme scolaire.
- **Bulletins Automatiques** : Génération en un clic (Format PDF professionnel).

### 👨‍🏫 Pour les Enseignants
- **Saisie Rapide des Notes** : Interface optimisée pour gagner du temps.
- **Appel Numérique** : Suivi rigoureux des présences et retards.
- **Gestion des Devoirs** : Publication des travaux à faire pour les élèves.

### 👨‍👩‍👧 Pour les Parents (Module Premium)
- **Suivi en Temps Réel** : Consultation des notes et absences de leurs enfants.
- **Accès Mobile** : Interface responsive adaptée aux smartphones.

### 👑 Pour le Superadmin (Propriétaire du SaaS)
- **Gestion des Écoles** : Activation/Désactivation des accès.
- **Suivi des Revenus** : Dashboard financier (Paiements en FCFA).
- **Gestion des Licences** : Contrôle strict des abonnements.

## 🛠️ Technologies Utilisées
- **Backend** : PHP 8.x (Architecture API REST / PDO)
- **Frontend** : Bootstrap 5, JavaScript (ES6+), SweetAlert2
- **Base de Données** : MySQL (InnoDB)
- **Serveur** : Apache / Nginx

## 💰 Modèle Business (Afrique de l'Ouest)
Le SaaS propose deux plans adaptés au contexte local :
- **FREE** : 0 FCFA (Jusqu'à 50 élèves, idéal pour tester).
- **PREMIUM** : 5 000 FCFA / mois (Élèves illimités + Module Parents + Bulletins PDF).
